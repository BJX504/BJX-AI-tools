# 生物统计 AI 代码生成工具

## 项目简介

这是一个专为生物统计和公卫数据分析设计的AI辅助工具，基于Python和Streamlit构建。该工具允许用户上传数据文件，输入分析需求，然后通过iFlow API调用大模型生成相应的Python代码，并自动运行代码展示结果。

## 功能特点

- **数据上传与预览**：支持CSV、XLSX和TXT格式的文件，自动显示数据集信息和基本统计
- **生物统计专用分析**：支持描述性统计、相关性分析、方差分析、回归分析、生存分析等多种生物统计方法
- **AI代码生成**：通过iFlow API调用GLM-5模型生成高质量的Python代码
- **代码运行与结果展示**：一键运行生成的代码，自动显示统计结果和图表
- **数据导出**：支持将分析结果导出为CSV、Excel和JSON格式
- **API密钥管理**：API Key一次配置永久使用，用户无需每次输入

## 技术栈

- **前端**：Streamlit
- **数据处理**：pandas
- **统计分析**：scipy、statsmodels
- **数据可视化**：seaborn、matplotlib
- **API接口**：iFlow API

## 快速开始

### 环境准备

1. 克隆本项目到本地
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
3. 在`config.py`文件中配置您的iFlow API Key：
   ```python
   IFLOW_API_KEY = "your_iFlow_api_key_here"
   ```

### 运行项目

```bash
streamlit run app.py
```

然后在浏览器中打开 http://localhost:8501 即可使用。

## 使用指南

1. **上传数据**：在侧边栏选择要分析的数据文件
2. **选择分析类型**：从下拉菜单中选择适合的生物统计分析类型
3. **输入分析需求**：在文本框中详细描述您的分析需求，或点击"自动生成需求"按钮
4. **生成代码**：点击"生成代码"按钮，等待AI生成Python代码
5. **运行代码**：点击"运行代码"按钮，查看分析结果和图表
6. **导出数据**：根据需要将结果导出为不同格式

## 部署选项

### 本地部署

直接在本地运行`streamlit run app.py`命令。

### 云端部署

1. 将项目上传到GitHub仓库
2. 登录Streamlit Cloud
3. 连接GitHub仓库并部署应用
4. 在Streamlit Cloud的应用设置中配置环境变量

## 注意事项

- 请确保您的iFlow API Key有效且有足够的配额
- 对于大型数据集，可能需要更长的处理时间
- 生成的代码仅供参考，复杂分析可能需要手动调整
